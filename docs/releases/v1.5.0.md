# MailDraft v1.5.0

## Highlights

- Added tags for drafts, templates, memos, and content blocks, including reuse from existing tag suggestions and count-based single-tag filtering.
- Added content blocks for managing short reusable fragments and inserting them into draft openings, bodies, and closings.
- Added draft-to-template conversion and stronger variable preset recommendations based on recent use and matching tags.
- Added custom right-click menus across list views for faster duplicate, restore, delete, and conversion actions.

## Notes

- Runtime data still uses SQLite. Existing `1.4.0` data continues to load, and legacy JSON storage still migrates on first launch when present.
- Windows releases remain unsigned and may show the standard warning on first launch.
